# 超级管理员账号管理功能快速上手指南

## 快速访问

1. 使用超级管理员账号登录系统
2. 在顶部导航栏右侧点击"超管账号管理"
3. 或直接访问：`http://your-domain/Staff/SuperAdminAccountManagement`

## 主要功能

### 📊 查看统计信息
页面顶部显示三个统计卡片：
- **总超级管理员数** - 系统中所有超级管理员的总数
- **激活超级管理员** - 当前激活状态的超级管理员数量
- **本月新增** - 本月新创建的超级管理员数量

### ➕ 添加超级管理员
1. 点击右上角"添加超级管理员"按钮
2. 填写以下信息：
   - **用户名**（必填）：用于登录的账号名
   - **密码**（必填）：登录密码，系统会自动加密
   - **姓名**（必填）：真实姓名
   - **激活状态**：勾选表示账号激活，可以登录系统
3. 点击"保存"按钮完成添加

### ✏️ 编辑超级管理员
1. 在超级管理员列表中找到要编辑的账号
2. 点击该行的"编辑"按钮
3. 修改信息（注意：密码无法在此修改）
4. 点击"保存"按钮保存修改

### 🗑️ 删除超级管理员
1. 在超级管理员列表中找到要删除的账号
2. 点击该行的"删除"按钮
3. 在确认对话框中点击"确定"
4. ⚠️ **注意**：如果该账号有关联数据，建议使用"禁用"功能而非删除

### 🔍 搜索和筛选
**搜索框：**
- 输入用户名或姓名的部分内容
- 支持模糊匹配

**状态筛选：**
- 全部状态：显示所有超级管理员
- 激活：只显示激活状态的账号
- 禁用：只显示禁用状态的账号

点击"搜索"按钮应用筛选条件

### 📥 导出数据
1. 可以先使用搜索和筛选功能过滤数据
2. 点击"导出数据"按钮
3. 系统会生成 CSV 文件并自动下载
4. 文件名格式：`超级管理员数据_时间戳.csv`

## 导出的文件内容

CSV 文件包含以下列：
- 超级管理员ID
- 用户名
- 姓名
- 创建日期
- 最后登录日期
- 账号状态（激活/禁用）

文件特点：
- UTF-8 编码，支持中文
- 表头清晰，可直接用 Excel 打开
- 日期格式：`YYYY-MM-DD HH:MM:SS`

## 界面说明

### 数据列表
| 列名 | 说明 |
|------|------|
| ID | 超级管理员的唯一标识符 |
| 用户名 | 登录用户名 |
| 姓名 | 真实姓名 |
| 创建日期 | 账号创建时间 |
| 最后登录 | 最近一次登录时间，如果从未登录显示"从未登录" |
| 状态 | 绿色徽章表示"激活"，灰色徽章表示"禁用" |
| 操作 | "编辑"和"删除"按钮 |

### 分页导航
- 显示当前页的数据范围和总数
- 每页显示 20 条记录
- 可以点击页码直接跳转
- 使用"上一页"/"下一页"按钮翻页

## 常见问题

### Q: 忘记超级管理员密码怎么办？
A: 密码无法在编辑界面修改。需要通过数据库直接修改或联系系统管理员。

### Q: 为什么删除超级管理员失败？
A: 可能该账号有关联的操作记录。建议使用"禁用"功能，取消"激活状态"勾选。

### Q: 导出的 CSV 文件在哪里？
A: 文件会自动下载到浏览器的默认下载文件夹。

### Q: 可以批量删除超级管理员吗？
A: 当前版本不支持批量操作，需要逐个删除或禁用。

### Q: 超级管理员和普通管理员有什么区别？
A: 
- **超级管理员**：拥有系统最高权限，可以管理所有功能包括管理员账号和超级管理员账号
- **普通管理员**：根据分配的权限管理特定功能模块

## 安全提示

### ⚠️ 重要注意事项
1. 只有超级管理员可以访问此功能
2. 所有操作都会被记录到系统日志中
3. 密码会自动加密，无法查看原始密码
4. 建议定期导出数据备份
5. 删除前请确认账号没有关联数据

### 🔒 密码安全
- 密码使用 SHA256 加密算法
- 密码不会以明文形式存储或显示
- 建议使用强密码（包含大小写字母、数字和特殊字符）

### 📝 操作日志
所有操作都会记录日志，包括：
- 添加超级管理员
- 更新超级管理员信息
- 删除超级管理员
- 导出数据

可以在"日志管理"中查看详细的操作记录。

## 快速操作示例

### 示例 1：添加新的超级管理员
```
1. 点击"添加超级管理员"按钮
2. 输入用户名：superadmin2
3. 输入密码：SecurePassword@123
4. 输入姓名：张三
5. 勾选"激活状态"
6. 点击"保存"
```

### 示例 2：搜索特定超级管理员
```
1. 在搜索框输入：张
2. 点击"搜索"按钮
3. 系统显示所有姓名包含"张"的超级管理员
```

### 示例 3：禁用超级管理员
```
1. 找到要禁用的超级管理员
2. 点击"编辑"按钮
3. 取消勾选"激活状态"
4. 点击"保存"
5. 该账号将无法登录系统
```
