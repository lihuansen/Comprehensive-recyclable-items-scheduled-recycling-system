# 基地工作人员账号管理快速使用指南

## 功能入口

登录基地工作人员工作台后，在顶部导航栏可以看到：

```
[基地管理]  [账号管理]  ◇ 基地工作台 ◇
```

点击"账号管理"即可进入个人中心。

## 主要功能

### 1️⃣ 个人中心
- 查看账户基本信息
- 显示注册时间和最后登录时间
- 提供两个主要操作：
  - 📝 编辑个人信息
  - 🔒 修改密码

### 2️⃣ 编辑个人信息
可以修改以下信息：
- ✏️ 姓名（必填）
- 📱 手机号码（必填，格式验证）
- 🪪 身份证号
- 💼 职位
- 🏢 工作站
- ⭐ 专业特长
- ⏰ 班次类型（必填，下拉选择：白班/夜班/早班/中班/晚班/轮班）

### 3️⃣ 修改密码
- 🔑 输入当前密码
- 🆕 输入新密码（至少6个字符）
- ✅ 确认新密码
- ⚠️ 修改成功后需要重新登录

## 操作流程

### 编辑信息流程
```
个人中心 → 点击"编辑信息" → 修改需要更新的字段 → 保存修改 → 返回个人中心
```

### 修改密码流程
```
个人中心 → 点击"修改密码" → 输入当前密码 → 输入新密码 → 确认新密码 → 提交 → 重新登录
```

## 注意事项

⚠️ **重要提示**
1. 手机号必须是有效的中国大陆手机号（11位，1开头）
2. 姓名、手机号、班次类型为必填项
3. 修改密码后系统会自动退出，需要使用新密码重新登录
4. 所有更改会立即生效并保存到数据库

## 安全说明

🔒 **安全保障**
- 所有表单都有防伪令牌保护
- 密码修改需要验证当前密码
- 密码以加密形式存储
- 修改密码后强制重新登录
- 所有操作都需要登录身份验证

## 常见问题

**Q: 忘记当前密码怎么办？**  
A: 请联系系统管理员重置密码。

**Q: 可以修改用户名吗？**  
A: 不可以，用户名是唯一标识，不能修改。

**Q: 修改信息后需要重新登录吗？**  
A: 编辑个人信息不需要重新登录，但修改密码需要。

**Q: 密码有什么要求？**  
A: 密码至少需要6个字符。

**Q: 可以修改基地信息吗？**  
A: 基地名称和基地ID等信息由管理员分配，无法自行修改。
